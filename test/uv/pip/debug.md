# pip debug

## Unsupported command

<!-- from pip_debug.rs::debug_warn -->

The `pip debug` command is unsupported - users should use `uvx pip debug` instead.

```console
$ uv pip debug
success: false
exit_code: 2
----- stdout -----

----- stderr -----
error: pip's `debug` is unsupported (consider using `uvx pip debug` instead)
```
