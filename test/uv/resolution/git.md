# Git Dependency Resolution

Tests for resolving Git dependencies across all uv commands (compile, install, sync, lock, freeze).

```toml
[test]
required-features = ["git"]
```

## Basic git+https dependency

<!-- from pip_compile.rs::compile_git_https_dependency -->

Resolve a package from a Git repository using HTTPS.

```toml title="requirements.in" snapshot=true
uv-public-pypackage @ git+https://github.com/astral-test/uv-public-pypackage
```

```console
$ uv pip compile requirements.in
success: true
exit_code: 0
```

```toml title="uv.filter" snapshot=true

filters = [
  { regex = "@[0-9a-f]{40}", replacement = "@[COMMIT]" }
```
