# Resolution Strategies

Tests for resolution strategies (`--resolution`) and upgrade behavior.

## Upgrade behavior

### No upgrade

<!-- from pip_compile.rs::upgrade_none -->

By default, existing pinned versions are preserved.

```toml
# file: requirements.in
black==23.10.1
```

```toml
# file: requirements.txt
black==23.10.1
click==8.1.2
    # via black
mypy-extensions==1.0.0
    # via black
packaging==23.2
    # via black
pathspec==0.11.0
    # via black
platformdirs==4.0.0
    # via black
```

```console
$ uv pip compile requirements.in --output-file requirements.txt
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in --output-file requirements.txt
black==23.10.1
    # via -r requirements.in
click==8.1.2
    # via black
mypy-extensions==1.0.0
    # via black
packaging==23.2
    # via black
pathspec==0.11.0
    # via black
platformdirs==4.0.0
    # via black

----- stderr -----
Resolved 6 packages in [TIME]
```

## Lowest resolution

### Lowest-direct with sdist

<!-- from pip_compile.rs::compile_sdist_resolution_lowest -->

The `--resolution=lowest-direct` strategy pins direct dependencies to their lowest versions.

```toml
# file: requirements.in
anyio @ https://files.pythonhosted.org/packages/2d/b8/7333d87d5f03247215d86a86362fd3e324111788c6cdd8d2e6196a6ba833/anyio-4.2.0.tar.gz
```

```console
$ uv pip compile requirements.in --resolution=lowest-direct --python-version 3.12
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in --resolution=lowest-direct --python-version 3.12

    # via -r requirements.in
idna==3.6
    # via anyio
sniffio==1.3.1
    # via anyio

----- stderr -----
Resolved 3 packages in [TIME]
```

### Lowest-direct with fork

<!-- from pip_compile.rs::lowest_direct_fork_min_python -->

Lowest-direct respects Python version markers in forks.

```toml
# file: requirements.in
pycountry >= 22.1.10
setuptools >= 50.0.0 ; python_version >= '3.12'
```

```console
$ uv pip compile requirements.in --universal --resolution lowest-direct

    # via
    #   -r requirements.in
    #   pycountry
setuptools==69.2.0 ; python_full_version < '3.12'

    #   -r requirements.in
    #   pycountry

Resolved 3 packages in [TIME]
```

## Sdist resolution

### Lowest-direct with sdist URL

<!-- from pip_install.rs::install_sdist_resolution_lowest -->

The `--resolution=lowest-direct` strategy with a source distribution URL installs the exact version.

```toml
# file: requirements.in
anyio @ https://files.pythonhosted.org/packages/2d/b8/7333d87d5f03247215d86a86362fd3e324111788c6cdd8d2e6196a6ba833/anyio-4.2.0.tar.gz
```

```console
$ uv pip install -r requirements.in --resolution=lowest-direct
success: true
exit_code: 0
----- stdout -----

----- stderr -----
Resolved 3 packages in [TIME]
Prepared 3 packages in [TIME]
Installed 3 packages in [TIME]
 + anyio==4.2.0 (from https://files.pythonhosted.org/packages/2d/b8/7333d87d5f03247215d86a86362fd3e324111788c6cdd8d2e6196a6ba833/anyio-4.2.0.tar.gz)
 + idna==3.6
 + sniffio==1.3.1
```

## Preferences

### Preference with trailing zero normalization

<!-- from pip_compile.rs::compile_types_pytz -->

Resolve a package with a preference that omits a trailing zero. The existing pinned version
`types-pytz==2021.1` should be normalized to `types-pytz==2021.1.0` in the output.

```toml
# file: requirements.in
types-pytz
```

```toml
# file: requirements.txt
types-pytz==2021.1
```

```console
$ uv pip compile requirements.in -o requirements.txt
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in -o requirements.txt
types-pytz==2021.1.0
    # via -r requirements.in

----- stderr -----
Resolved 1 package in [TIME]
```

## Universal resolution

### Prefer upper bounds in universal resolution

<!-- from pip_compile.rs::universal_prefer_upper_bounds -->

When resolving with `--universal`, prefer upper bounds to satisfy conflicting requirements.

```toml
# file: requirements.in
pylint < 3 ; sys_platform == 'darwin'
pylint > 2 ; sys_platform != 'darwin'
```

```console
$ uv pip compile requirements.in --python-version 3.8 --universal
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in -p 3.8 --universal
astroid==2.15.8
    # via pylint
colorama==0.4.6 ; sys_platform == 'win32'
    # via pylint
dill==0.3.8
    # via pylint
isort==5.13.2
    # via pylint
lazy-object-proxy==1.10.0
    # via astroid
mccabe==0.7.0
    # via pylint
platformdirs==4.2.0
    # via pylint
pylint==2.17.7
    # via -r requirements.in
tomli==2.0.1 ; python_full_version < '3.11'
    # via pylint
tomlkit==0.12.4
    # via pylint
typing-extensions==4.10.0 ; python_full_version < '3.11'
    # via
    #   astroid
    #   pylint
```

### Remove unnecessary Python version markers

<!-- from pip_compile.rs::universal_unnecessary_python -->

When resolving with `--universal`, unnecessary Python version markers are removed.

```toml
# file: requirements.in
iniconfig ; python_version >= '3.7'
```

```console
$ uv pip compile requirements.in --python-version 3.8 --universal
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in -p 3.8 --universal
iniconfig==2.0.0
    # via -r requirements.in

----- stderr -----
warning: The requested Python version 3.8 is not available; 3.12.[X] will be used to build dependencies instead.
Resolved 1 package in [TIME]
```

### Requires Python prefetch

<!-- from pip_compile.rs::requires_python_prefetch -->

```
exclude-newer = "2025-01-01T00:00:00Z"

```

```toml
# file: requirements.in
voluptuous<=0.15.1
```

```console
$ uv pip compile requirements.in
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv --cache-dir [CACHE_DIR] pip compile requirements.in
voluptuous==0.14.2
    # via -r requirements.in

----- stderr -----
Resolved 1 package in [TIME]
```
