## Basic export

### Export to requirements.txt format

<!-- from export.rs::requirements_txt_no_hashes -->

Export a project's dependencies to requirements.txt format without hashes.

```toml
# file: pyproject.toml
[project]
name = "project"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["anyio==3.7.0"]

[build-system]
requires = ["setuptools>=42"]
build-backend = "setuptools.build_meta"
```

```console
$ uv lock
success: true
exit_code: 0
```

```console
$ uv export --no-hashes
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv export --cache-dir [CACHE_DIR] --no-hashes
-e .
anyio==3.7.0
    # via project
idna==3.6
    # via anyio
sniffio==1.3.1
    # via anyio

----- stderr -----
Resolved 4 packages in [TIME]
```

## Virtual project export

### Export virtual project

<!-- from export.rs::virtual_empty -->

Exporting a virtual project with no dependencies shows a warning.

```toml
# file: pyproject.toml
[tool.uv.workspace]
members = []
```

```console
$ uv lock
success: true
exit_code: 0
----- stdout -----

----- stderr -----
warning: No `requires-python` value found in the workspace. Defaulting to `>=3.12`.
Resolved in [TIME]
```

```console
$ uv export --frozen --no-hashes
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv export --cache-dir [CACHE_DIR] --frozen --no-hashes

----- stderr -----
warning: `uv export` is a no-op for virtual workspaces with no members
```

## Export dependency groups

### Export with dependency group

<!-- from export.rs::virtual_dependency_group -->

Export a dependency group from a virtual project.

```toml
# file: pyproject.toml
[tool.uv.workspace]
members = []

[dependency-groups]
dev = ["anyio"]
```

```console
$ uv lock
success: true
exit_code: 0
----- stdout -----

----- stderr -----
warning: No `requires-python` value found in the workspace. Defaulting to `>=3.12`.
Resolved 4 packages in [TIME]
```

```console
$ uv export --frozen --no-hashes --group dev
success: true
exit_code: 0
----- stdout -----
# This file was autogenerated by uv via the following command:
#    uv export --cache-dir [CACHE_DIR] --frozen --no-hashes --group dev
anyio==4.3.0
    # via workspace (group: dev)
idna==3.6
    # via anyio
sniffio==1.3.1
    # via anyio

----- stderr -----
```
