## Project extras

Exporting with project-defined optional extras.

```toml

[project]
name = "project"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["typing-extensions"]

[project.optional-dependencies]
async = ["anyio==3.7.0"]

[build-system]
requires = ["setuptools>=42"]
```
