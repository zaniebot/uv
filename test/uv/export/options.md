## No editable with environment variable

<!-- Derived from [`export::no_editable_env_var`](https://github.com/astral-sh/uv/blob/main/crates/uv/tests/it/export.rs#L4615-L4640) -->

Using the `UV_NO_EDITABLE` environment variable to disable editable installs.

```toml
# mdtest

[environment]
env = { UV_NO_EDITABLE = "1" }
```

## Only group and extra conflict

<!-- Derived from [`export::export_only_group_and_extra_conflict`](https://github.com/astral-sh/uv/blob/main/crates/uv/tests/it/export.rs#L4643-L4692) -->

Using `--only-group` and `--extra` together should error.

```toml
# file: pyproject.toml

[project]
name = "project"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = []

[project.optional-dependencies]
test = ["pytest"]

[dependency-groups]
dev = ["ruff"]
```

```console
$ uv export --only-group dev --extra test
success: false
exit_code: 2
----- stdout -----

----- stderr -----
error: the argument '--only-group <ONLY_GROUP>' cannot be used with '--extra <EXTRA>'

Usage: uv export --only-group <ONLY_GROUP> --exclude-newer <EXCLUDE_NEWER>

For more information, try '--help'.
```

```console
$ uv export --only-group dev --all-extras
success: false
exit_code: 2
----- stdout -----

----- stderr -----
error: the argument '--only-group <ONLY_GROUP>' cannot be used with '--all-extras'

Usage: uv export --only-group <ONLY_GROUP> --exclude-newer <EXCLUDE_NEWER>

For more information, try '--help'.
```

## Lock workspace mismatch with frozen

<!-- Derived from [`export::export_lock_workspace_mismatch_with_frozen`](https://github.com/astral-sh/uv/blob/main/crates/uv/tests/it/export.rs#L4696-L4734) -->

The members in the workspace and in the lockfile mismatch.

```toml
# file: pyproject.toml

[project]
name = "foo"
version = "0.1.0"
requires-python = ">=3.12"
```

```toml
# file: uv.lock

version = 1
revision = 3
requires-python = ">=3.12"

[[package]]
name = "bar"
version = "0.1.0"
source = { virtual = "." }
dependencies = []
```

```console
$ uv export --frozen
success: false
exit_code: 2
----- stdout -----

----- stderr -----
error: The lockfile at `uv.lock` needs to be updated, but `--frozen` was provided: Missing workspace member `foo`. To update the lockfile, run `uv lock`.
```

## Multiple packages

<!-- Derived from [`export::multiple_packages`](https://github.com/astral-sh/uv/blob/main/crates/uv/tests/it/export.rs#L4738-L4878) -->

Export multiple packages within a workspace.

```toml
# file: pyproject.toml

[project]
name = "root"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["foo", "bar", "baz"]

[tool.uv.sources]
foo = { workspace = true }
bar = { workspace = true }
baz = { workspace = true }

[tool.uv.workspace]
members = ["packages/*"]
```

```toml
# file: packages/foo/pyproject.toml

[project]
name = "foo"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["anyio"]
```

```toml
# file: packages/bar/pyproject.toml

[project]
name = "bar"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["typing-extensions"]
```

```toml
# file: packages/baz/pyproject.toml

[project]
name = "baz"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = ["iniconfig"]
```
